### Labour Connect
- This is web based application, thanks connects ``Manual Laboured`` persons with clients in need of their services.

```bash
    - This services include;
        - Plumber
        - Carpenters
        - painter
        - Mechanics etc.
```
- This application is built with the following technologies:
    - **Front-End**
        - HTML5
        - CSS3
        - Javascript
    - **Back-End**
        - php
        - MySQL database
